Avant-propos

Avertissement : ce thème n'est pas conçu pour migrer un site existant. Si vous avez déjà un site avec du contenu, vous devrez réécrire tout le contenu sur ce nouveau site.

Pensez donc bien à sauvegarder vos données et votre contenu avant d'entamer l'installation du thème Pokefolio.

En effet, le thème possède des types de contenu personnalisé qui n'est pas présent sur les autres thèmes ou sur WordPress par défaut. De plus l'installation remet tous les compteur à zéro.

Démarrer

Installer le thème

Après avoir téléchargé le fichier sur notre site web, il vous faudra installer l'extension All-in-One WP Migration sur votre site WordPress. Il sera probablement requis d'avoir l'extension qui permet d'installer des fichiers plus gros.

Une fois que l'extension est installée, cliquez sur son onglet et allez dans le sous-onglet "Importer".

Ici vous pouvez glisser/déposer le fichier du thème Pokefolio ou bien cliquer sur "Importer depuis" et retrouver le fichier sur votre disque dur.

L'extension vous demandera si vous voulez écraser la base de donnée, cliquez sur "Oui".

A partir de maintenant il ne reste plus qu'à attendre que l'extension installe le site.

Les fonctionnalités du thème

Types de contenu

Dans cette section vous retrouverez les différents types de contenu qui rempliront votre site, leur rôle et où vous pouvez y accéder.

Articles (blog)

Les articles constituent la partie blog de votre site. C'est ici que vous pourrez écrire des articles en rapport avec votre métier, ou sur n'importe quel sujet.

Vous pouvez créer, dupliquer, et éditer des articles.

Il est important de penser à mettre une image mise en avant ansi qu'un extrait de l'article (excerpt) afin que son affichage sur le site soit optimal.

Vous pouvez vous aider de l'extension Yoast SEO pour optimiser le référencement lorsque vous rédigez un article.

Expériences

Le type de contenu Expériences liste vos expériences, par exemple, là où vous avez travaillé, le rôle que vous y avez rempli, ansi que les accomplissements réalisés.

Les expériences s'affichent sur la page d'accueil.

Services

Ce type de contenu sert à décrire les différents services que vous proposez et s'affiche sur la page "Services" de votre site ansi que sur la page d'accueil.

Skills

Ici vous pouvez créer un post différent pour chacune de vos faculté. Il s'agit d'un article qui décrit l'un de vos talents.

Testimonials

Cette partie sert à entrer les témoignages de vos clients à propos de votre travail. Les testimonials s'affichent sur la page d'accueil sous forme de slider.

Works

Ici vous pouvez parler des travaux que vous avez effectué et ainsi montrer ce que vous savez faire.

En bas de la page vous devez indiquer quelques informations à propos du travail que vous avez effectué. Ces informations s'afficheront en haut de la page de l'article final.

Vous devez indiquer le client pour lequel vous avez effectué ce travail, les différents services que vous lui avez rendu, ansi que l'année.

Champs personnalisés (Custom Field Suite)

Les champs personnalisés permettent de rajouter des informations supplémentaires sur les pages.

Par défaut ces champs ont déjà été créés en étant adaptés aux besoin du thème, vous n'avez donc pas grand chose à faire ici, sauf si vous voulez toucher au code du thème en lui même.
